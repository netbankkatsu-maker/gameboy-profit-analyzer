## 🐛 既知の問題と推奨対応

### 1. 日本語文字エンコーディング (優先度: 中)

**症状**: 日本語タイトルが `\udcef\uff7f\uff7d...` のようなガベージ文字で表示される

**根本原因**: 
- Node.js/Express でのリクエストボディの文字エンコーディング設定
- または Firebase Firestore への保存時の変換問題

**推奨修正**:
1. backend/server.js の bodyParser設定確認:
   ```javascript
   app.use(express.json({ limit: '50mb' }));
   app.use(express.urlencoded({ limit: '50mb', extended: true }));
   ```

2. クライアント側のリクエストヘッダ確認:
   ```javascript
   headers: {
     'Content-Type': 'application/json; charset=utf-8'
   }
   ```

3. Firebase Firestore のコレクションシーマ確認

**テスト方法**:
```bash
curl -X POST http://localhost:3000/api/analyze \
  -H "Content-Type: application/json; charset=utf-8" \
  -d '{"title":"テスト","askingPrice":10000}'
```
